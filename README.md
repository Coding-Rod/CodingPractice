# Practice Web

## pug

### Install pug

~~~bash
sudo npm install pug-cli -g
npm install --save-dev gulp-pug-3
pug -V
~~~

### Run pug

~~~bash
pug -w ./<input directory> -o ./<output directory> -P
~~~

## sass

### Install sass

~~~bash
sudo apt-get install -y sass
sass --version
~~~

### Run sass

~~~bash
sass -w ./<input directory>:./<output directory>
~~~
