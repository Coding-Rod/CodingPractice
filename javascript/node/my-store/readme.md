### Middlewares

Los Middlewares se pueden utilizar como pipes, es decir la salida de uno es la entrada del entro, se ejecutan entre el request y el response y sus casos de uso son los siguientes:

- Validar datos
- Capturar errores
- Validar permisos
- Controlar accesos
