# Angular

## Verifica versión de Node

~~~bash
node -v
~~~

## Verifica versión de npm

~~~bash
npm -v
~~~

## Instala el CLI de Angular

~~~bash
npm -g @angular/cli
~~~

## Verifica tu instalación

~~~bash
ng version
~~~

## Crea tu primer proyecto

~~~bash
ng new my-project
~~~

## Ejecuta el servidor de desarrollo

~~~bash
ng serve Dentro de la carpeta de tu proyecto.
~~~
