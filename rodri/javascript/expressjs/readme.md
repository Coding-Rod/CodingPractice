## REST

Respresentational State Transfer

- Post
- Get
- Put
- Delete

### Ejemplo

|Method|/productos|/products/{id}|
|:----:|:--------:|:------------:|
|GET   |Get list  |Get           |
|PUT   |Replace   |Update/Replace|
|PATCH |No Apply  |Update        |
|POST  |Create    |No apply      |
|DELETE|Delete    |Delete        |
